## How to Use This Tracker

1. **Add New Topics**: Create a new directory under `topics/` for each major learning area
2. **Daily/Weekly Logs**: Update this file with progress notes
3. **Resources**: Store related resources, notes, and exercises in topic directories
4. **Review**: Regularly review and update your progress
